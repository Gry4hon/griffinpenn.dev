Project Vulcn


Languages: C#
Tools used: Unity, Blender, Procreate, Oculus Rift S, Arduino

Project Vulcn was my capstone project for my senior year at Boise State University, and my on going research project. The project centers around a virtual reality Thermal Immersion Gauntlet™, which coupled with the game I developed around it, provides a unique immersive experiance for the player. The game itself is a resource management and tower defence style game that centers around a entity creation game mechanic, in which the player animates golems.

The goal with these golems is to defend and repair your crashed ship. These golems are animated with the help of the Vulcn Industries (fictional in game company) Animator Gauntlet, which combinding pieces of your crashed ship and piles of scrap, the player animates golems to assist with the exodus from the planet Egabrag One.
